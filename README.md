# 💻 Projeto

Este projeto foi desenvolvido para a avaliação formativa de Front-End com o objetivo de criar um site responsivo para celulares e tablets.

# 🚀 GalaxyTours

O GalaxyTours é uma agência de turismo espacial que vende passagens para diversos destinos ao redor do mundo como: Lua, Marte e Satúrno.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:
/
├── index.html
└── src/
    ├── styles/  (Estilos da página)
    ├── scripts/ (Lógica de interação)
    └── images/  (Ativos visuais)

## 🚀 Resolução
* A resolução do projeto foi feita utilizando media query definindo limites de tamanho de tela e alterações para esses limites
